RU:
Приведен пример реализации простого бота,
который будет искать возможности арбитража между парами BTC/ETH и ETH/USDT.

Бот постоянно проверяет, какая валютная пара в данный момент является наиболее прибыльной для торговли. 
Это очень простой бот, и вы можете добавить множество дополнительных функций, таких как совершение сделок,
просмотр торговой истории, использование различных стратегий и т.д.

ENG:
An example of the implementation of a simple bot is given 
that will look for arbitrage opportunities between the BTC/ETH and ETH/USDT pair.

The bot constantly checks which currency pair is currently the most profitable for trading. 
This is a very simple bot and you can add a lot of additional functionality such as executing trades, 
viewing trading history, using different strategies, etc.
